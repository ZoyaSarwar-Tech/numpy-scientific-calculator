# 🧮 NumPy Scientific Calculator

A beginner-friendly Scientific Calculator built using **Python** and **NumPy**.  
This project performs mathematical, trigonometric, statistical, and array operations.

---

# 🚀 Features

- Addition
- Subtraction
- Multiplication
- Division
- Power Calculation
- Square Root
- Trigonometric Functions
  - Sin
  - Cos
  - Tan
- Array Operations
- Mean Calculation
- Standard Deviation
- Maximum & Minimum Values
- Rounded Values

---

# 🛠 Technologies Used

- Python
- NumPy
